# ESP32-S3 音乐盒物料清单（实际配置版）

## 基于实际持有硬件整理

本 BOM 与 BOM_V2.md（展示盒版本）的主要区别：使用实际持有的 ESP32-S3-DevKitC-1 开发板、德力普 12V 10800mAh 锂电池，喇叭规格为 4Ω 11W 高音 + 4Ω 10W 全频。

---

## 电子元器件

### 主控模块

| 序号 | 名称 | 型号/规格 | 数量 | 备注 |
|------|------|-----------|------|------|
| 1 | ESP32-S3-DevKitC-1 | N8R8（8MB Flash + 8MB PSRAM）| 1 | 已持有，主控开发板 |

| 序号 | 名称 | 型号/规格 | 数量 | 备注 |

| 序号 | 名称 | 型号/规格 | 数量 | 备注 |

## 与 BOM_V2.md 的对比

| 项目 | BOM_V2 | 本配置（实际硬件）|
|------|--------|-----------------|
| 主控 | ESP32-S3-WROOM-1 模块 | **ESP32-S3-DevKitC-1 开发板** |
| 功放 | ZK-502C 或 PAM8403 | **ZK-502C（TPA3116D2）** |
| DAC | 无（方案B含 MAX98357A）| **PCM5102A（新增，接 ZK-502C）**|
| 高音喇叭 | 8Ω 5W | **4Ω 11W** |
| 全频喇叭 | 4Ω 5-10W | **4Ω 10W** |
| 电源 | 12V DC 适配器 | **12V 10800mAh 锂电池** |
| NFC | PN532 | **PN532（相同）** |
| 显示屏 | GC9A01 1.28寸 | **GC9A01 1.28寸（相同）**|
